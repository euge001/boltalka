# План развития проекта (Roadmap)

Цель: Разделение проекта на голосовой чат (Болталка) и экспертную систему для кодинга (Coder Expert).

## Модуль 1: Coder Expert (Текущая задача)
**Принцип работы:** ИИ только слушает и моментально выводит решение в чат без голоса.

- [x] **Транскрипция в реальном времени:** Стриминг речи через `gpt-realtime-mini` (максимальная скорость, минимальная цена).
- [x] **Передача в спец-модель:** Автоматическая отправка распознанного текста в `chat.php` (модель GPT-4o или o1).
- [x] **Текстовый вывод:** Ответ эксперта выводится в чат в виде текста и кода с объяснениями.
- [x] **Запрет голоса:** Эксперт не генерирует аудио-ответы.
- [x] **Markdown:** Поддержка форматирования кода и текста в чате.

## Дальнейшие шаги (по запросу):
1. **Тонкая настройка стека:** Возможность передавать текущий стек технологий пользователя эксперту "на лету".
2. **Контроль задержек:** Оптимизация передачи текста между моделями для сокращения времени ожидания ответа.
3. **История чата:** Сохранение решений эксперта в интерфейсе.
